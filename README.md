# 化学濃度計算

日本語で使える、教育・研究補助向けの静的な濃度計算サイトです。

## 公開ページ

https://1874676733qqco-glitch.github.io/-/

## できること

- モル濃度調製: 目標濃度、最終体積、分子量、純度から必要質量を計算
- C1V1希釈: 原液濃度、目標濃度、最終体積から必要な原液量と希釈液量を計算
- 質量％濃度: w/v%、v/v%、wt% の調製量を計算
- 単位換算: M、mM、uM、nM、g/L、mg/mL、ug/mL、ppm を換算
- 混合計算: 複数溶液を混ぜた後の濃度を計算
- 連続希釈: 希釈倍率と段数から各段階の濃度表を作成

## 特徴

- サーバー不要の HTML/CSS/JavaScript だけで動作
- ブラウザで計算でき、スマートフォン幅にも対応
- 計算結果だけでなく計算過程も表示
- 計算履歴は使用中のブラウザ内だけに保存され、外部には送信されません

## 注意

このサイトは教育・研究補助用です。臨床、医療、危険物調製の正式手順としては使用しないでください。
